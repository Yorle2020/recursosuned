Curso: Fundamentos de Desarrollo Web

Este curso introduce los principios esenciales del desarrollo web: estructura con HTML, estilos con CSS, interactividad con JavaScript y buenas prácticas para publicar proyectos. Está diseñado para que el estudiante avance de conceptos básicos a la creación de una aplicación web funcional.
